# Чехлы на сиденья Renault Symbol

## Типы чехлов

### Универсальные (бескаркасные)

Надеваются на любое сиденье, фиксируются резинками и стяжками. Не закрывают боковины полностью. Цена — 1500–3000 ₽ за комплект. **Недостаток**: плохая фиксация, скользят, не защищают боковые подушки безопасности.

### Полукаркасные (полумодельные)

Шьются по лекалам Symbol с учётом профиля сиденья, но без жёсткой основы. Закрывают боковины, фиксируются лентами-липучками. Есть вариант с прорезями для боковых подушек безопасности. Цена — 4000–7000 ₽.

### Каркасные (модельные / кастомные)

Полностью повторяют геометрию штатных сидений. Устанавливаются съёмом обивки — сажаются на штатные пазы и липучки. Совместимы с подушками безопасности. Цена — 8000–15 000 ₽.

## Установка на передние сиденья

1. Снять подголовники (зажать фиксатор у основания, вытянуть вверх)
2. Надеть чехол на спинку сиденья, расправить по контуру
3. Затянуть нижние стяжки (под сиденьем, между спинкой и подушкой)
4. Надеть чехол на подушку, заправить края между подушкой и пластиковой облицовкой
5. Затянуть передние стяжки под сиденьем
6. Установить подголовники

> ⚠ При наличии боковых подушек безопасности использовать только чехлы с прорезями или со специальными ослабленными швами. Блокировка подушки грозит травмой.

## Установка на заднее сиденье

1. Снять подушку заднего сиденья — поддеть передний край вверх, вывести из двух защёлок
2. Надеть чехол на подушку, затянуть стяжки снизу
3. Установить подушку на место — защёлкнуть (проверить фиксацию)
4. Снять подголовники задних сидений (если есть и съёмные)
5. Надеть чехол на спинку, расправить, затянуть стяжки
6. В сложных чехлах — закрепить по периметру с помощью липучек к штатной обивке

## Рекомендации по уходу

- Тканевые и велюровые — сухая чистка пылесосом раз в месяц, химчистка раз в год
- Кожа / экокожа — протирать влажной салфеткой, использовать кондиционер для кожи каждые 3 месяца
- Не сушить чехлы на батарее — материал садится
- При курении в машине — ткань впитывает запах, лучше экокожа
