# Руководство по ремонту Renault Symbol

**Renault Symbol (Рено Симбол) — бюджетный субкомпактный седан B-сегмента, выпускавшийся с 1999 по 2021 год. Создан на платформе Renault Clio II, адаптированной для развивающихся рынков. На некоторых рынках известен как Renault Thalia.**

| Параметр | Значение |
|----------|----------|
| Производитель | Renault |
| Сборка | Турция (Oyak-Renault, Бурса), Россия (Автофрамос/Рено Россия, Москва), Аргентина (Santa Isabel), Бразилия (São José dos Pinhais), Марокко (Casablanca), Тайвань |
| Класс | Субкомпактный (B-сегмент) |
| Платформа | Renault A Platform (X65), позднее Nissan B (Step II) |
| Тип кузова | 4-дверный седан |
| Компоновка | Переднемоторная, передний привод |
| Колёсная формула | 4×2 |

## Тормозная система

| Узел | Характеристика |
|------|---------------|
| **Передние тормоза** | Дисковые, вентилируемые, диаметр 259 мм, толщина 20 мм |
| **Задние тормоза** | Барабанные, диаметр 203 мм (на части комплектаций с 1.6 16V — дисковые, диаметр 240 мм) |
| **Привод** | Гидравлический, двухконтурный, диагональный |
| **Вакуумный усилитель** | Диаметр 205 мм или 228 мм |
| **ABS** | Bosch 5.3 (ранние выпуски) / Bosch 8.0 (после 2005 г.) |
| **EBD** | Электронное распределение тормозных сил (с ABS Bosch 8.0) |
| **Передние колодки** | Толщина накладки 14 мм (новая), мин. 2 мм |
| **Задние колодки** | Толщина накладки 4,8 мм (новая), мин. 1,5 мм |

> ⚠ На части комплектаций для рынка РФ и Латинской Америки ABS могла отсутствовать (базовые версии). Задние дисковые тормоза устанавливались только на версии с двигателем K4M в максимальных комплектациях.

## Подвеска

| Ось | Тип |
|-----|-----|
| Передняя | Независимая, типа McPherson (стойка + треугольный рычаг со стабилизатором поперечной устойчивости) |
| Задняя | Полузависимая (балка скручивающегося типа со стабилизатором) |

> ⚠ Передняя подвеска унифицирована с Renault Clio II. Задняя балка — усиленная, под специфику седана.
