## Радиатор отопителя (печка)

Радиатор отопителя — это маленький радиатор системы охлаждения, расположенный в салоне за центральной консолью. Через него проходит горячая охлаждающая жидкость, а вентилятор продувает воздух в салон.

| Параметр | Значение |
|----------|----------|
| Материал | Алюминий + пластиковые бачки |
| Расположение | За центральной консолью, слева от блока управления |
| Соединение | 2 трубки к моторному щиту (слева) |
| Антифриз | Renault D / D+ (см. [жидкости](../spravochnaya/zhidkosti.md)) |
| Средний ресурс | 8–12 лет / 150 000–200 000 км |

### Признаки неисправности радиатора печки

| Симптом | Причина | Решение |
|---------|---------|---------|
| Холодный воздух при повороте на горячий | Закоксовался / забился | Промывка или замена |
| Запотевание стёкол, запах антифриза | Микротрещина | Замена радиатора |
| Лужа антифриза под бардачком | Разрыв бачка или трубки | Немедленная замена |
| Печка дует холодным только на пассажира | Воздушная пробка в системе | Прокачка охлаждения |

### Замена радиатора печки

```admonition warning
Замена радиатора отопителя на Symbol — трудоёмкая операция (8–12 часов). Требуется полный разбор передней панели. При появлении первых признаков течи не откладывайте.
```

1. Отключите АКБ, слейте антифриз
2. Снимите рулевое колесо (специальный съёмник)
3. Демонтируйте центральную консоль и блок управления отопителем
4. Открутите 4 винта крепления радиатора
5. Извлеките радиатор вместе с корпусом
6. Установите новый радиатор, залейте антифриз, удалите воздух

**Альтернатива (быстрый метод):** На некоторых версиях Symbol III можно заменить радиатор через вырез в корпусе без снятия панели.

## Моторчик вентилятора (отопителя)

Электродвигатель, создающий поток воздуха через радиатор печки и/или испаритель кондиционера.

| Параметр | Значение |
|----------|----------|
| Расположение | Под бардачком, справа |
| Тип | Электродвигатель постоянного тока с щётками |
| Мощность | ~120–150 Вт |
| Предохранитель | F5 (40A) под капотом |
| Средний ресурс | 10–15 лет |

### Скорости вентилятора

Скорости регулируются **резистором** (сопротивлением), установленным в кожухе вентилятора.

| Скорость | Сопротивление | Примечание |
|----------|---------------|------------|
| 1 (мин) | Максимальное | Слабый обдув |
| 2 | Среднее | |
| 3 | Малое | |
| 4 (макс) | **Прямое соединение** | Резистор не участвует |

**Типичная неисправность:** Перегорает резистор скоростей (особенно на 1–2 скорости) или его термопредохранитель. Симптом: вентилятор работает только на 4 скорости.

### Замена моторчика вентилятора

1. Открутите 3 винта под бардачком
2. Отсоедините разъём питания
3. Снимите моторчик (вынимается вниз)
4. При замене: очистите посадочное место, проверьте работу резистора

## Салонный фильтр

| Параметр | Значение |
|----------|----------|
| Расположение | Под бардачком, за пластиковой крышкой |
| Тип | Угольный / бумажный |
| Размер | 245×195×30 мм (стандарт) |
| Артикул | Original: 77 01 205 211 |
| Интервал замены | **Каждые 30 000 км или 2 года** |

```admonition info
Салонный фильтр часто игнорируют. Забитый фильтр снижает производительность печки, запотевают стёкла, в салоне запах пыли. Замена — 5 минут и ~300–500 руб.
```

### Замена салонного фильтра

1. Откройте бардачок, отожмите ограничитель (слева)
2. Опустите бардачок полностью
3. Снимите пластиковую крышку (защёлки по бокам)
4. Выньте старый фильтр (запомните направление стрелки!)
5. Вставьте новый фильтр стрелкой **вниз** (поток воздуха сверху вниз)
6. Закройте крышку, поднимите бардачок

## Блок управления отопителем

Типы блоков управления в зависимости от комплектации:

| Тип | Описание | Особенности |
|-----|----------|-------------|
| **Механический** | 3 ручки: температура, скорость, режим | Symbol I (1999–2002), база Symbol II |
| **Механический + кондиционер** | + кнопка A/C | Symbol II, Symbol III база |
| **Полуавтоматический** | Автоматическая температура + ручные режимы | Symbol II 2005+, III Optima |
| **Автоматический климат-контроль** | Полная автоматика + LCD дисплей | Symbol III Privilège |

### Расшифровка режимов

| Режим | Обозначение | Назначение |
|-------|-------------|------------|
| 🧊 | Стрелка в лицо | Охлаждение летом |
| 🦶🧊 | Стрелка в ноги + лицо | Умеренное охлаждение |
| 🦶 | Стрелка в ноги | Отопление, прогрев |
| 🦶🪟 | Ноги + стекло | Отопление с обдувом стёкол |
| 🪟 | Стекло | Против запотевания / обогрев |

## Типичные неисправности HVAC

| Проблема | Вероятная причина | Ремонт |
|----------|------------------|--------|
| Вентилятор не работает на 1–3 скорости | Сгорел резистор скоростей | Замена резистора (300–500 руб.) |
| Вентилятор не работает совсем | 1. Предохранитель F5<br/>2. Моторчик | 1. Заменить<br/>2. Замена моторчика |
| Течёт антифриз под бардачком | Трещина радиатора печки | Замена радиатора |
| Плохо греет с одной стороны | Воздушная пробка / забит радиатор | Прокачка / промывка |
| Посторонний шум при работе вентилятора | Лист / мусор в улитке | Разбор, чистка |
| Запотевание стёкол | 1. Забит салонный фильтр<br/>2. Рециркуляция включена | 1. Замена<br/>2. Выключить рециркуляцию |
| Пахнет антифризом из дефлекторов | Течь радиатора печки | Замена радиатора |
| Нет холодного воздуха (кондиционер) | См. [1.3 Кондиционер и климат-контроль](./expluataciya/1-3.md) | — |
| Заслонки не переключаются | Обрыв троса / поломка привода | Регулировка / замена привода |

## Советы по эксплуатации

```text
Летом:
- Перед включением кондиционера проветрите салон (откройте окна на 1 мин)
- Используйте рециркуляцию для быстрого охлаждения
- Раз в месяц включайте кондиционер зимой на 10 мин (для смазки компрессора)

Зимой:
- Режим «стекло» + кондиционер = быстрое осушение воздуха
- Выключайте рециркуляцию при запотевании
- Не включайте печку на максимум до прогрева двигателя

Всесезонно:
- Меняйте салонный фильтр раз в год
- При первых признаках запаха антифриза — немедленная диагностика
```
