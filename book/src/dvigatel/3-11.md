# 3.11 Снятие и установка двигателя

Пошаговая инструкция по снятию двигателя K7J/K7M с Renault Symbol. Работа выполняется с помощником. Двигатель снимается в сборе с КПП или отдельно.

## Подготовка

### Инструменты

| Инструмент | Назначение |
|-----------|------------|
| Подъёмник или яма | Обязательно — снизу много операций |
| Домкрат + опоры (козлы) 4 шт | Для фиксации авто |
| Домкрат трансмиссионный | Для поддержки КПП/двигателя |
| Набор головок E-Torx T40–T60 | ГБЦ, опоры ДВС |
| Набор головок Torx T20–T55 | Впуск, выпуск, стартер |
| Головки 6-гранные 8–24 мм | Основной крепёж |
| Динамометрический ключ 5–150 Н·м | Обязательно — моменты критичны |
| Съёмник шкива коленвала | Для снятия шкива |
| Фиксатор маховика | Для откручивания болта коленвала |
| Ветошь, перчатки, очки | |

### Подготовительные работы

1. **Отключите АКБ:** снимите минусовую клемму
2. **Слейте масло:** открутите сливную пробку поддона (ключ на 16)
3. **Слейте ОЖ:** открутите сливную пробку радиатора (слева снизу) и пробку на блоке цилиндров (за выпускным коллектором)
4. **Снимите капот:** пометьте положение петель маркером — 4 болта Torx T40
5. **Снимите АКБ** с площадкой и отодвиньте в сторону

## Отключение систем

### Охлаждение

- Снимите расширительный бачок (1 болт + шланг)
- Отсоедините патрубки радиатора (верхний и нижний)
- Снимите радиатор с вентилятором (2 болта M8 сверху, отщёлкнуть снизу)
- Отсоедините патрубки отопителя (в моторном отсеке, слева)

### Топливная система

```admonition danger
Топливо легко воспламеняется! Работайте в хорошо проветриваемом помещении. Перед отсоединением магистралей сбросьте давление: заглушите двигатель, подождите 10 минут, ослабьте заглушку топливной рампы.
```

- Отсоедините обратную магистраль (быстросъём)
- Отсоедините подающую магистраль (ключ на 13)
- **Заглушите оба штуцера** — чистым болтом М12 или заглушкой
- Снимите топливную рампу с форсунками (2 болта M8)

### Электрика

| Разъём | Расположение | Примечание |
|--------|-------------|------------|
| ЭБУ (ECU) | Справа, за АКБ | 48-контактный, фиксатор отщёлкнуть |
| Жгут двигателя | По левому лонжерону | Главный разъём (3 секции) |
| Масса двигателя | На корпусе КПП | 1 болт M10 |
| Масса кузова | На левом лонжероне | 1 болт M10 |
| Генератор (B+) | На задней стороне ген. | Ключ на 13 |
| Стартер (клемма 50) | На втягивающем реле | Ключ на 8 |
| Лямбда-зонды | На приёмной трубе | 2 шт, ключ на 22 |

### Выпускная система

1. Снимите теплозащитный экран (4 болта Torx T20)
2. Открутите выпускной коллектор от ГБЦ — 8 гаек M8 (ключ на 13), работайте торцевой головкой
3. Отсоедините приёмную трубу от коллектора — 2 пружинных болта M10
4. Снимите лямбда-зонд из коллектора (если мешается)

### Навесное оборудование

- **Компрессор кондиционера:** открутите 4 болта M8, отведите в сторону (не разъединяйте магистрали!)
- **Генератор:** снимите ремень, открутите 2 болта M10
- **Насос ГУР:** снимите ремень, открутите 3 болта M10, отведите в сторону
- **Стартер:** 2 болта M10 + провод клеммы 50

## Демонтаж двигателя

### Снятие КПП (вариант с разделением)

Двигатель можно снимать в сборе с КПП (тяжелее, ~140 кг) или отдельно. Для снятия отдельно:

1. Снимите колёса и приводные валы ([4.3 Приводные валы](../transmissiya/4-3.md))
2. Поддомкратьте КПП домкратом
3. Открутите 6 болтов M10 крепления КПП к двигателю
4. Снимите стартер (освобождает доступ к нижним болтам)
5. Аккуратно разъедините КПП и двигатель (отодвиньте КПП назад)
6. Опустите КПП на подставку

### Извлечение двигателя

1. Зацепите двигатель талью за штатные проушины:
   - Передняя проушина — справа спереди ГБЦ (M10)
   - Задняя проушина — на корпусе помпы (M8)
2. Натяните трос тали
3. Открутите **правую опору** ДВС (3 болта M10 к кузову, 2 болта M10 к двигателю)
4. Открутите **левую опору** (на КПП, 2 болта M10)
5. Открутите **заднюю опору** (под КПП, 1 болт M12 + 2 болта M10)
6. Убедитесь, что все шланги, провода и тросы отсоединены
7. Поднимите двигатель на 5–10 см, проверьте — ничего не держит?
8. Извлекайте вверх через капотный проём, с наклоном вправо (сторона радиатора)

```admonition warning
Вес двигателя K7J с навесным — ~120 кг. Работайте только с помощником и талью. Не поднимайте двигатель за впускной коллектор или поддон — они не рассчитаны на вес.
```

## Установка

Установка — в обратной последовательности. Обязательно замените:

| Деталь | Причина |
|--------|---------|
| Прокладка ГБЦ | При снятой ГБЦ — обязательно |
| Прокладка впускного коллектора | Одноразовая |
| Прокладка выпускного коллектора | Одноразовая |
| Прокладка поддона | При снятии поддона |
| Сальник коленвала (передний) | Пока двигатель снят |
| Сальник КПП | Со стороны двигателя |
| Помпа + ремень ГРМ | Удобный момент |

### Моменты затяжки при установке

| Соединение | Момент, Н·м |
|------------|-------------|
| Опоры ДВС к кузову | 45 ± 5 |
| Опоры ДВС к двигателю | 45 ± 5 |
| КПП к двигателю (M10) | 50 ± 5 |
| Выпускной коллектор к ГБЦ (M8) | 20 ± 2 |
| Приёмная труба к коллектору | 25 ± 3 |
| Генератор (M10) | 40 ± 4 |
| Компрессор кондиционера | 25 ± 3 |
| Шкив коленвала | 100 ± 5 (K7J) / 110 ± 5 (K7M) |
| Стартер | 35 ± 5 |

### После установки

1. **Заливка масла:** 3,2 л (K7J) / 3,5 л (K7M) — проверьте уровень
2. **Заливка ОЖ:** 6,2 л — с удалением воздушной пробки ([см. 3.4](./3-4.md#Удаление-воздушной-пробки))
3. **Подключение АКБ:** проверьте полярность
4. **Запуск:** первый запуск — без нагрузки, дайте прогреться на ХХ до 90°C
5. **Проверка на течи:** осмотрите все соединения (масло, ОЖ, топливо)
6. **Считывание ошибок:** после первого запуска проверьте сканером OBD2

```admonition tip
Первый запуск после снятия двигателя может быть долгим (топливная система развоздушивается). Включите зажигание на 5–7 секунд, выключите, повторите 2–3 раза — насос поднимет давление. Затем заводите с открытым дросселем (педаль газа в пол до половины).
```
