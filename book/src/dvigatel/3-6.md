# 3.6 Замена ремня ГРМ

Пошаговая инструкция по замене ремня привода ГРМ (газораспределительного механизма) на двигателях K7J, K7M, K4J, K4M. Двигатели интерференционного типа — **обрыв ремня гарантированно гнёт клапаны** и требует капитального ремонта ГБЦ.

```admonition danger
Соблюдайте регламент замены: **60 000 км или 4 года** (что наступит раньше).
Пробег — не гарантия: резина стареет, ремень может лопнуть и на 40 000 км, если машина простояла 6 лет.
```

## Схема привода ГРМ

```mermaid
flowchart LR
    subgraph Привод
        C[Коленвал] -->|зубчатый шкив| B[Ремень ГРМ]
        B -->|ведёт| CM[Распредвал]
        B -->|ведёт| WP[Помпа]
        B --> TNR[Натяжной ролик]
        B --> TNS[Успокоитель]
    end

    subgraph Навесное
        ALT[Генератор] -->|поликлиновой ремень| C
        AC[Кондиционер *] -.->|опционально| C
    end

    style C fill:#1565c0,color:#fff
    style CM fill:#e65100,color:#fff
    style WP fill:#2e7d32,color:#fff

    note[N"K4J/K4M: 2 распредвала (DOHC)"]
```

**K7J / K7M** — SOHC (один распредвал, 8 клапанов).
**K4J / K4M** — DOHC (два распредвала, 16 клапанов). Процедура замены идентична, разница в количестве меток.

## Признаки износа ремня ГРМ

| Признак | Описание | Срочность |
|---------|----------|-----------|
| Трещины на внешней стороне | Видны при осмотре, резина «шагрень» | **Немедленная замена** |
| Расслоение зубьев | Зубья крошатся, выкрашиваются | **Немедленная замена** |
| Масло на ремне | Просачивание через сальник коленвала / распредвала | Замена ремня + сальников |
| Посторонний шум (свист/скрип) | Шум натяжного ролика или помпы | Диагностика — возможен износ подшипников |
| Превышение пробега / возраста | >60 000 км или >4 года | **Плановая замена** |

## Необходимые инструменты и запчасти

| Позиция | Для K7J/K7M (8V SOHC) | Для K4J/K4M (16V DOHC) |
|---------|----------------------|----------------------|
| **Ремень ГРМ** | Gates 5438XS / Conti CT995 | Gates 5429XS / Conti CT1013 |
| **Натяжной ролик** | SKF VKM 34010 | SKF VKM 34012 |
| **Помпа** | SKF VPC 1035 / Hepu P156 | SKF VPC 1036 / Hepu P159 |
| **Болты помпы** | 4 шт, в комплекте | 3 шт, в комплекте |
| **Сальник коленвала** | 32×52×8 мм | 32×52×8 мм |
| **Сальник распредвала** | 36×48×7 мм | 35×50×8 мм (2 шт) |
| **Герметик** | Loctite 5910 / Victor Reinz | Loctite 5910 / Victor Reinz |
| **Охлаждающая жидкость** | ~5,5 л (слив при замене помпы) | ~5,8 л (слив при замене помпы) |

> **Рекомендация:** Покупайте **комплект ГРМ** (ремень + ролик + помпа). Отдельная покупка деталей часто выходит дороже.

### Инструменты

| Инструмент | Назначение |
|------------|------------|
| Головка на 10, 13, 16, 18, 21 | Основные крепления |
| Torx T50 | Шкив коленвала (болт) |

| Ёмкость для слива антифриза | ~6 л |

## Порядок замены

### 1. Подготовка

1. Отсоедините клемму **«–»** АКБ.
2. Снимите **защиту картера** (если есть).
3. Слейте **антифриз** (через кран на блоке и пробку радиатора).
4. Снимите **воздушный фильтр** в сборе с корпусом.
5. Снимите **правое переднее колесо** и **подкрылок**.
6. Снимите **верхнюю крышку ремня ГРМ** (3 болта на 10).

### 2. Доступ к ремню

1. Ослабьте натяжение **поликлинового ремня** генератора (открутите болт натяжителя, сдвиньте ролик).
2. Снимите поликлиновой ремень.
3. Снимите **шкив коленвала** (головка T50, болт может быть затянут 180–200 Н·м — используйте фиксатор маховика).
4. Снимите **нижнюю крышку ремня ГРМ** (2 болта).

### 3. Установка ВМТ (верхняя мёртвая точка)

**Для K7J/K7M (8V):**
1. Проверните коленвал по часовой стрелке за болт шкива до совмещения меток.
2. **Метка маховика:** треугольник на маховике совпадает с прорезью на корпусе КПП (смотровой лючок слева).
3. **Метка распредвала:** шкив распредвала — отверстие совпадает с меткой на задней крышке ГРМ.
4. Оба поршня 1-го и 4-го цилиндра — в ВМТ.

**Для K4J/K4M (16V):**
1. Коленвал — метка на шкиве/зубчатом колесе совпадает с меткой на масляном насосе.
2. Распредвалы — метки на звёздочках совпадают с метками на задней крышке (обе звёздочки).
3. Зафиксируйте распредвалы специальным приспособлением (или ровным металлическим стержнем ∅6 мм через отверстия).

### 4. Демонтаж старого ремня

1. Ослабьте **натяжной ролик** (шестигранник 6 мм, поверните против часовой стрелки).
2. Снимите ремень ГРМ.
3. Осмотрите **ролик** — проверьте люфт, шум при вращении (даже малый люфт — замена).
4. Открутите **помпу** — 3–4 болта (Torx T30 или головка на 10).
5. Очистите привалочную плоскость помпы от остатков прокладки.

### 5. Установка новой помпы

1. Нанесите герметик на новую прокладку помпы (тонкий валик по внутреннему периметру).
2. Установите помпу, затяните болты моментом **10–12 Н·м** (крест-накрест).
3. Установите **новый натяжной ролик** (болт 25 Н·м).

### 6. Установка нового ремня

1. **Убедитесь** что метки коленвала и распредвала(ов) совпадают.
2. Наденьте ремень: начните со **шкива коленвала**, затем **помпа**, затем **натяжной ролик**, затем **распредвал(ы)**.
3. Натяните ремень, вращая натяжной ролик **по часовой стрелке**.
4. **Правильное натяжение:** ремень не провисает между шкивами, при повороте на 90° пальцем — умеренное усилие.

Для **K7J/K7M (8V)**: стрелка на натяжном ролике должна совпасть с прорезью (автоматический натяжитель).
Для **K4J/K4M (16V)**: натяжение контролируется индикатором на ролике.

### 7. Контрольная проверка

1. **Проверните коленвал на 2 оборота вручную** (ключом на болт коленвала).
2. Проверьте совпадение **всех меток**.
3. Если метки не совпали — повторите шаг 5–6.
4. Убедитесь, что ремень правильно лежит на всех шкивах (не смещён к краю).

### 8. Сборка

1. Установите **нижнюю крышку ремня ГРМ**.
2. Установите **шкив коленвала**, затяните болт моментом **180–200 Н·м**.
3. Установите **поликлиновой ремень** генератора.
4. Установите **верхнюю крышку ремня ГРМ**.
5. Установите **подкрылок** и **колесо**.
6. Залейте **антифриз**.
7. Подключите **АКБ**.
8. Заведите двигатель, проверьте на слух — посторонние шумы недопустимы.

### 9. Проверка после запуска

| Проверка | Норма | Действие при отклонении |
|----------|-------|------------------------|
| Шум ГРМ | Равномерный, без стуков | Проверить натяжение, если стук — остановить |
| Работа двигателя | Устойчиво на ХХ | ДТЛ тест-драйв |
| Индикатор Check Engine | Не горит | Диагностика OBD2 |
| Утечки антифриза | Нет | Подтянуть хомуты, проверить помпу |
| Уровень антифриза | Между MIN/MAX | Долить, выгнать воздушную пробку |

### 10. Рекомендации по обкатке

- Первые **500 км** — без высоких оборотов (>4000), без буксировки.
- Проверьте натяжение ремня после 1000 км (при повторном осмотре).
- При появлении свиста — ремень может быть перетянут, ослабьте натяжной ролик на 5°.

## Типовые ошибки

| Ошибка | Последствие | Как избежать |
|--------|-------------|-------------|
| Несовпадение меток на 1 зуб | Плавают обороты, потеря мощности, ошибка P0016 | Вращать коленвал на 2 оборота и перепроверять |
| Перетяжка ремня | Шум подшипников, обрыв ремня | Использовать пружинный индикатор на ролике |
| Старый болт помпы | Течь антифриза | Заменить на новые (в комплекте) |
| Не заменён сальник | Течь масла через неделю | Заменить сальник при первой замене ГРМ |
| Пропуск замены помпы | Через 30 000 км заклинит помпа → обрыв ремня | Менять помпу вместе с ремнём всегда |

## Сложность и время

| Операция | Время |
|----------|-------|
| Замена ремня ГРМ (с опытом) | 2–3 часа |
| Замена ремня ГРМ (новичок) | 4–6 часов |
| + замена помпы | +1 час |
| + замена сальников | +0,5 часа |
| + промывка системы охлаждения | +0,5 часа |

```admonish info "Замена помпы"
На двигателях K7J/K7M помпа приводится ремнём ГРМ и расположена слева от блока.
На K4J/K4M помпа также приводится ремнём ГРМ. **Обязательно меняйте помпу при каждой замене ремня!**
Экономия на помпе при текущей замене — ложная: заклинившая помпа порвёт ремень через 20–40 тыс. км.
```
